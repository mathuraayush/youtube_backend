# Vidora Frontend

Modern React + Vite frontend for the Vidora video platform backend.

## Stack
- React + Vite (JavaScript)
- Redux Toolkit + RTK Query
- Tailwind CSS
- shadcn-style component primitives
- Dark mode toggle

## Run locally
```bash
cd frontend
npm install
npm run dev
```

Backend base URL is preconfigured to:
`https://vidora-ic3e.onrender.com/api/v1`
